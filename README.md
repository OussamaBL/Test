La commande git init crée un nouveau dépôt Git.
La commande git status affiche l'état du répertoire de travail et de la zone de staging.
La commande git branch sert à créer, à répertorier, à renommer et à supprimer des branches.
La commande git remote vous permet de créer, d'afficher et de supprimer des connexions avec d'autres dépôts.
La commande Git Push ajoute les contenus d'un dépôt local vers un dépôt central.
La commande Git Diff compare les modifications dans les fichiers, les branches et les commits.
La commande Git add ajoute un changement dans le répertoire de travail à la zone de staging.
La commande git reset est un outil complexe et polyvalent pour annuler les changements.
La commande git pull est utilisée pour faire un fetch du contenu d'un dépôt distant et pour le télécharger, puis pour mettre à jour immédiatement le dépôt local qui correspond à ce contenu.
